# Таск 9 (Alert Project)

## Структура

* main.py -> Главный вход в программу происходит в файле main.py. Там создается движок и определяются основные правила для alert ошибок
* engine.py -> Движок для системы оповещения, запускает проверки и в случае чего принимает решение сделать ALERT
* rules -> Хранит python файлы с классами проверок (у каждого класса-правила есть метод check)
* virtual -> Хранит абстрактные классы
* ingestors -> Классы которые возвращают df из какого либо источника (http запрос или чтение csv файла например)
