# Выполнил: Кусков Арсений

## Данный курс позволяет познакомиться со всеми необходимыми инструментами Apache Spark. В данном курсе разбираются такие вопросы как:

- Основы Apache Spark и PySpark;

- Типы обрабатываемых файлов. Достоинства и недостатки;

- Join и Shuffle;

- Что такое Data Skew;

- Работа памяти в Apache Spark;

- Best Practices при работе со Spark;

- Delta Lake.
